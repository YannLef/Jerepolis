-----------------— 1) INITIALISATION -----------------—

--------------—— 1) INITIALISATION --------------——

Initialise le clavier : Chaque touche nécessaire dans le programme correspond à un booléen. Le booléen est égal à 1 si la touche est relachée Le booléen est égal à 0 si la touche est appuyée Le clavier doit être transmit dans les différents fichiers qui en ont besoin en utilisant le mot clé "extern" Initialise les callbacks utilisés par glut et permettant de gérer le clavier.
